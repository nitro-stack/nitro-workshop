# Reference links

Here is the list of everything we used in this workshop, as a reference:

  • Node.js as the base platform for everything
  • Angular with @azure/ng-deploy for the frontend
  • NestJS with @nestjs/azure-func-http for the backend
  • Azure Functions for API deployment
  • Azure Storage to host the website
  • Azure CLI to manage resources
  • Azure Functions Core Tools to test and deploy your backend

You should also take a look at Hexa, a great CLI companion to create and deploy resources on Azure interactively.
